# 📈 1️⃣ Decision Tree Regression

## 🔍 Overview
Decision Tree Regression is used to predict continuous numerical values.  
In this project, we build a regression model using DecisionTreeRegressor to predict target values based on input features.

## ⚙️ Steps Performed

- Data Loading  
- Data Preprocessing  
- Feature Selection  
- Model Training  
- Model Evaluation  
- Visualization of Results  

## 🛠️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn  

## 📊 Evaluation Metrics

- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- R² Score  

## 🚀 How to Run

Install required libraries:

pip install numpy pandas matplotlib scikit-learn

Open the notebook:

jupyter notebook decision_tree_regression.ipynb

# 🌳 What is a Decision Tree?

A Decision Tree is a supervised machine learning algorithm used for both:

✔ Regression (predict continuous values)  
✔ Classification (predict categories)  

It splits the dataset into branches based on feature values to make predictions.
